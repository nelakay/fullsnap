# FullSnap - Full Page Screenshot

A free Chrome extension that captures full-page screenshots with built-in cropping and PDF export.

## Features

- **Full page capture** — automatically scrolls and stitches the entire page
- **Crop tool** — drag handles to select exactly the area you want
- **Export as PNG** — full resolution download
- **Export as PDF** — auto-paginated A4 output
- **Copy to clipboard** — paste directly into docs, chat, etc.
- **Zoom controls** — fit to screen or zoom in for detail
- **100% free** — no subscriptions, no premium tiers, no data collection

## Install from source

1. Clone this repository
2. Open `chrome://extensions` in Chrome
3. Enable **Developer mode** (top right)
4. Click **Load unpacked**
5. Select the cloned folder

## Privacy

FullSnap processes everything locally in your browser. No data is collected or transmitted. See [Privacy Policy](PRIVACY_POLICY.md).
